Rancangan UI — Integrasi Tarikan × Sistem Pembayaran

Mockup interaktif · klik tab & kartu untuk melihat perubahan form

Alur yang dipahami

  1. Petugas buka Penarikan → Tambah Data
  2. Pilih siswa → saldo tabungan tampil otomatis
  3. Pilih tujuan: Pembayaran atau Lainnya
  4. Jika Pembayaran: pilih jenis tagihan dari sistem pembayaran (API)
  5. Isi nominal (bisa auto dari tagihan) → Simpan
  6. Sistem mencatat tarikan di e-Tabs dan mengirim pembayaran ke sistem pembayaran

Catatan integrasi

  • • Transaksi Lainnya = seperti sekarang (tanpa API pembayaran)
  • • Transaksi Pembayaran = dual-write (tabungan + API bayar)
  • • Jika API gagal → tarikan dibatalkan / rollback
  • • Kolom tabel penarikan: badge tujuan + jenis pembayaran
